SVG CON CSS

Vicente de Vidts & Daniel Carmona

En esta página les mostraremos distintas maneras de aplicar css con svg a través de la manipulación de distintos iconos sobre la higiene personal, utilizando animaciones, transiciones e incorporando elementos externos.

Una animación con CSS

En la siguiente animación se aplica translate y rotate. De esta manera vemos como el gatillo se aprieta y el desinfectante se esparce. Además se modifica la opacidad de las gotas de desinfectante para que estas después de cierta distancia desaparezcan.

Una transición con CSS

Las transiciones nos permiten interactuar con los iconos. De esta manera en este ejemplo podemos observar el contenido de la botella, al colocar el cursor sobre ella.

Una transición con CSS y algo más
